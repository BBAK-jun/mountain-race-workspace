# 🏔️ 등산복 입고 뛰어 — 제품 PRD

> "캐릭터 얼굴을 커스텀하고 랜덤 이벤트로 서로 경쟁하는 걸 보는 등산 레이싱 게임"

---

## 1. 제품 개요

| 항목        | 내용                                                                 |
| ----------- | -------------------------------------------------------------------- |
| 제품명      | 등산 레이싱 (Mountain Race)                                          |
| 한 줄 요약  | 친구 얼굴 넣고, 랜덤 스킬과 개억까 이벤트로 서로 망하는 걸 보는 게임 |
| 장르        | 자동 진행 레이싱 / 파티 관전 게임                                    |
| 플랫폼      | 웹 브라우저 (모바일/데스크톱)                                        |
| 플레이어 수 | 2~8인                                                                |
| 플레이 타임 | 1판 약 60~90초                                                       |
| 조작        | 없음 (관전 전용)                                                     |
| 타겟 유저   | 친구/동료 모임에서 가볍게 즐기고 싶은 2030 세대                      |
| 핵심 가치   | 플레이어 개입 없이 "결과를 보는 재미"에 집중하는 관전형 파티 게임    |

---

## 2. 배경 및 목적

### 2.1 문제 정의

- 모임/회식에서 다 같이 즐길 수 있는 가벼운 파티 게임이 부족
- 기존 파티 게임은 조작이 필요하거나, 설치가 번거로움
- 친구 얼굴을 넣어 웃기는 상황을 만드는 바이럴 콘텐츠에 대한 수요 존재

### 2.2 목적

- 브라우저에서 즉시 실행 가능한 무설치 파티 게임 제공
- 얼굴 합성 + 랜덤 이벤트로 자연스러운 바이럴 루프 생성
- 관전 중심 설계로 비게이머도 쉽게 참여 가능

---

## 3. 핵심 원칙

| 원칙        | 설명                                                 |
| ----------- | ---------------------------------------------------- |
| 완전 평등   | 모든 캐릭터 능력치 동일. 승패는 100% 확률 + 이벤트   |
| 억까 = 재미 | 1등일수록 방해를 많이 받아 역전 드라마가 자동 생성됨 |
| 관전 중심   | 플레이어 조작 없이 결과를 보는 재미에 집중           |
| 즉시 시작   | 닉네임 입력 → 시작. 최소한의 설정으로 빠르게 플레이  |

---

## 4. 사용자 흐름

```text
[랜딩페이지] → [설정 화면] → [레이스 관전] → [결과 화면]
                  ↑                             |
                  └────── 다시 하기 ←──────────┘
```

### 4.1 랜딩페이지

- Three.js(R3F)로 구현한 3D 배경
- 게임 타이틀 텍스트 + 간단한 설명 문구
- `게임 시작` 버튼 클릭 시 설정 화면으로 이동

### 4.2 설정 화면

1. 기본 4명 캐릭터 자동 생성, 최대 8명까지 추가 가능
2. 각 캐릭터마다 다음을 설정한다.

- 닉네임 입력
- 얼굴 이미지 업로드
- 미업로드 시 기본 이모지 랜덤 배정
- 고유 컬러 자동 배정
- Three.js(R3F) 3D 캐릭터 실시간 미리보기

3. 맵 선택은 MVP 기준 `기본 산길` 1종 고정
4. 최소 2명 이상이면 `등산 시작` 활성화

### 4.3 레이스 관전

1. 카운트다운 후 출발
2. 3D 산악 코스에서 캐릭터들이 자동으로 경주
3. 우측 상단 순위 리스트 표시
4. 하단 진행률 바 표시
5. 전역 이벤트와 피살기 발동 시 화면 중앙 대형 알림 표시
6. 약 60초 내 완주

### 4.4 결과 화면

1. Three.js(R3F) 3D 포디엄 연출
2. 전체 순위 표시
3. MVP 시상
4. `다시 하기` / `로비로` 버튼 제공
